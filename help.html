Google Meet Formatting Help


Formatting:



Bold: Use **text** to get text
Italics: Use *text* to get text
Underline: Use _text_ to get text
Strikethrough: Use ~~text~~ to get text
Superscript: Use ^text^ to get text
Spoiler: Use ||text|| to get

text


Code: Use `text` to get text
Quote: Use \"\"text\"\" to get
text

Highlighting: Use #text# to get text

Commands:


Use /tableflip to replace your message with (╯°□°）╯︵ ┻━┻
Use /unflip to replace your message with ┬─┬ ノ( ゜-゜ノ)
Use /shrug to replace your message with ¯\_(ツ)_/¯
Use /mute @person to hide all messages from the person until you reload the page
Use /unmute @person to show all messages from the person who you muted previously

Other:


Mentions: Use @name to send a private message to anyone with "name" in their name
Images: Use $$ Image address $$ to insert an image into the chat
